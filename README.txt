# Loja Paola Gamer — Fullstack Completo (5 abas)
## Backend
cd backend
mvnw spring-boot:run
API: http://localhost:8080
H2: http://localhost:8080/h2-console (jdbc:h2:mem:testdb / user: sa)
## Frontend
Abra frontend/index.html (recomendado VS Code + Live Server).
Fluxo: Login -> Cadastro -> Loja -> Carrinho -> Pagamento.
Saudação dinâmica por gênero e busca funcionando.
Imagens online (Unsplash) — pode trocar por arquivos locais em frontend/img/.
